# Build Error Pattern Memory — src111_merge15

패치 목적: Javadoc/블록 주석 내부에 예시 표기용 `'/* ... */'`가 포함되어 조기 `*/`이 발생, 주석이 비정상 종료되며 파서가 붕괴되는 문제 해결.

해결 방법: 문제 시퀀스의 말미 `*/`를 `*&#47;`(슬래시 HTML 엔티티)로 치환하여, 주석은 안전하게 유지되고 문서 렌더링은 동일하게 보이도록 처리.

대상 패턴:

- `/* ... */` (공백 유무 모두)
- `/ * ... */` (간헐적 공백 포함)
- `/* … */` (유니코드 줄임표)
